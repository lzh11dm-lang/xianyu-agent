# 咸鱼运营 Agent - MVP 版本设计

## 产品定位
对话式 AI 投流运营助手，通过自然语言交互，自动执行咸鱼投流 SOP。

## MVP 核心功能
1. **对话式界面** - 输入指令，AI 理解后执行
2. **浏览器连接** - 连接比特浏览器
3. **曝光计划创建** - 按 SOP 自动创建（出价20元，日预算40元）

## UI 设计风格

### 视觉风格
- **主题**：深色科技风（契合 AI/运营工具属性）
- **配色**：
  - 背景：#0A0A0F（纯黑）
  - 主色：#00F0FF（电光青）
  - 次色：#BF00FF（霓虹紫）
  - 文字：#FFFFFF / #A0A0A0

### 布局结构
```
┌─────────────────────────────────────┐
│  顶部导航栏（连接状态 + 账号切换）   │
├─────────────────────────────────────┤
│                                     │
│         消息列表区域                 │
│    (用户消息 + AI回复 + 执行结果)    │
│                                     │
├─────────────────────────────────────┤
│  快捷指令区（可折叠）                │
├─────────────────────────────────────┤
│  输入框 + 发送按钮                   │
└─────────────────────────────────────┘
```

### 消息类型
1. **用户消息** - 右侧，蓝色渐变背景
2. **AI思考消息** - 左侧，灰色背景，显示 AI 正在分析
3. **AI回复消息** - 左侧，带 Markdown 格式
4. **执行结果消息** - 左侧，带执行状态图标

### 快捷指令
- 🔗 连接浏览器
- 📊 创建曝光计划
- 📋 查看投流状态
- 🔄 刷新数据

## 技术实现

### 前端
- Expo 54 + React Native
- TailwindCSS 样式
- 流式 SSE 输出（实时显示 AI 回复）

### 后端
- Express.js
- 比特浏览器 API 集成
- AI 对话处理
